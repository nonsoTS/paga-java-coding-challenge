API Endpoints

POST /api/planted

Submit planting data

Example JSON:

{
  "farmName": "MyFarm",
  "season": "Spring 2024",
  "cropType": "Corn",
  "plantingArea": 10.5,
  "expectedAmount": 50.0
}

POST /api/harvested

Submit harvest data

Example JSON:

{
  "farmName": "MyFarm",
  "season": "Spring 2024",
  "cropType": "Corn",
  "actualAmount": 45.5
}

Testing the API

Use tools like Postman, curl, or your browser's developer tools to test the API.

Example curl command:

curl -X POST http://localhost:8080/api/planted \
  -H "Content-Type: application/json" \
  -d '{"farmName":"MyFarm","season":"Spring 2024","cropType":"Corn","plantingArea":10.5,"expectedAmount":50.0}'
